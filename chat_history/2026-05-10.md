## Přidané funkce (chronologicky)

### start.sh
Bash skript kontrolující prerekvizity:
1. Docker daemon běží, compose dostupný
2. Porty 8989/9000/8080/8081 volné
3. Diskové místo ≥ 2 GB
4. Zastavení existujících kontejnerů
5. Build + start + čekání na healthy (max 120s)
6. Curl ping na každý /health

### PostgreSQL persistenci (contacts-cpp)
- `postgres:16-alpine` — port 5433 na hostu (5432 byl obsazený)
- `contacts-cpp` přepsán na `libpqxx` connection pool (8 spojení)
- Schema auto-create + seed data při startu
- Volume `postgres_data`

```sql
CREATE TABLE contacts (id TEXT PRIMARY KEY, first_name TEXT NOT NULL, ...)
CREATE TABLE phone_numbers (id BIGSERIAL, contact_id TEXT REFERENCES contacts ON DELETE CASCADE, ...)
```

### Nové C++ endpointy
- `GET /dedup?threshold=` — Levenshtein similarity O(n²)
- `GET /analytics` — kategorie, domény emailů, labely telefonů
- `GET /export/vcard` + `POST /import/vcard` — vCard 3.0
- `GET /db/tables` — raw PostgreSQL tabulky
